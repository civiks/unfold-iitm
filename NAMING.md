# The name

- **unfold** — it started as the quiz unfolder (every question on one page). Kept
  the name as it grew into a portal declutter extension.
- **No "IITM" in the name** — it's unofficial. IITM is in the repo name
  (`unfold-iitm`) and description, not the name itself.
- **`[unfold]`** — brackets left over from the `[placeholder]` stage, kept as is.
